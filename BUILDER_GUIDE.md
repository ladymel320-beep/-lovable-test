## 🚀 How to Use the Builder

### Getting Started
1. Visit **http://localhost:5173/** for the marketing site
2. Click **"Start Building"** or **"Get Started"** to open the builder
3. The builder opens at **http://localhost:5173/builder**

### Building Your Website

**Step 1: Add Components**
- Click any component in the left panel to add it to the canvas
- OR drag components onto the canvas

**Step 2: Edit Content**
- Click any component on the canvas to select it
- Edit text directly by clicking on it
- Use the property editor (right panel) to customize styles

**Step 3: Customize Styles**
- Select a component
- Change colors, fonts, sizes in the right panel
- See changes instantly!

**Step 4: Export**
- Click **"Export HTML"** button (top right)
- Copy the HTML code or download as a file
- Open the file in any browser

## 🎨 Component Types Explained

### Heading (📝)
- Large title text
- Customizable font size, weight, color
- Perfect for page titles

### Text (📄)
- Paragraph content
- Multi-line text editing
- Full typography controls

### Button (🔘)
- Call-to-action button
- Customizable colors and size
- Great for links and actions

### Image (🖼️)
- Image placeholder
- Shows where images will go
- Customize size and spacing

### Container (📦)
- Section wrapper
- Can hold other components (future)
- Control background and spacing

### Spacer (📏)
- Add vertical spacing
- Adjustable height
- Helps with layout
